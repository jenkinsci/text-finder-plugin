Specify a regular expression using the syntax supported by the Java Pattern class to find future display name of this job.
If "your_regex.*" matches the line, then "your_regex" will be removed from the line and the rest of it wll be set as displayName of build.
Last found is served and is affected by search termination from main "Regular expression". Warning, spaces meter!
eg:
build id declaration:
        `Version: `
log:
        Version: 15.25.alpha-pre5
set display name of run:
        15.25.alpha-pre5
